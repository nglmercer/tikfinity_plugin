# browserapi

To install dependencies:

```bash
bun install
```

To execute or build:

```json
  "scripts": {
    "dev": "bun run src/index.ts",
    "build": "bun run build.ts"
  }
```

This project was created using `bun init` in bun v1.3.8. [Bun](https://bun.com) is a fast all-in-one JavaScript runtime.
